# Fullstack Notes App

**Frontend:** React, TypeScript, Redux Toolkit, RTK Query, MUI.  
Deployed on https://fullstack-notes-app-six.vercel.app/

**Backend:** Node.js, Nest.js, TypeScript, TypeORM, PostgreSQL.  
Deployed on https://fullstack-notes-app-f9yt.onrender.com

## To run Frontend locally

1. Go to `/client` directory.
2. Create `.env.development.local` file and `PORT` environment variable. (_For example: PORT=3001_).
3. Run `npm install` and `npm start` commands in terminal.

## To run Backend locally

1. Go to `/server` directory.
2. Create `.env` file and `DB_URL` environment variable. (_For example: DB_URL=postgresql://postgres:root@localhost:5432/notes_db_).
3. Run `npm install` and `npm run start:dev` commands in terminal.
